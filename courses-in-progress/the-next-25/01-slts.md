# SLTs: The Next 25 — A Gimbalabs Retrospective-to-Generative Course

## Module 1: What We Built

- 1.1: I can categorize Gimbalabs' completed and active projects by theme and structure.
- 1.2: I can identify recurring patterns across Gimbalabs projects.

## Module 2: What's Missing

- 2.1: I can compare Gimbalabs' project portfolio against our primary driver.
- 2.2: I can assess which project categories are over-represented or absent from our history.

## Module 3: Unfinished Business & Unrealized Potential

- 3.1: I can trace the evolution of a Gimbalabs project across funding rounds.
- 3.2: I can distinguish between projects that ended because they were complete and projects that ended because resources ran out.

## Module 4: Beyond Catalyst

- 4.1: I can map Gimbalabs' funding history and its concentration risks.
- 4.2: I can propose alternative sustainability models for a Gimbalabs project that don't depend on Project Catalyst.
- 4.3: I can design a project that generates value for the community without requiring external funding to start.

## Module 5: What the Ecosystem Needs That Nobody's Building

- 5.1: I can identify unsolved problems in the Cardano ecosystem that Gimbalabs is positioned to address.
- 5.2: I can evaluate a project idea against Gimbalabs' primary driver and the community's demonstrated strengths.

## Module 6: Designing the Next 25

- 6.1: I can draft a project proposal using the 11-point framework.
- 6.2: I can scope a project for a small team that can ship something meaningful within 3 months.
- 6.3: I can sequence a set of projects so that early projects create momentum that later projects depend on.

## Module 7: Building in Public (Optional)

- 7.1: I can design a project documentation strategy that makes our work legible and replicable by communities outside Gimbalabs.
- 7.2: I can define success metrics for a project that measure community impact beyond milestone delivery.
