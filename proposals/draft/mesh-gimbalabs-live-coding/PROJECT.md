## Proposal

### 1. Title or Brief Description
Mesh X Gimbalabs - Live Coding Sessions

### 2. Background or Rationale
In 2024, Mesh & Gimbalabs teamed up to host live coding sessions, inviting newcomers as well as veterans to learn to build with Mesh, hosted by Gimbalabs contributors. We would love to continue the live coding sessions.

Recordings from Live Coding sessions in 2024:
https://www.youtube.com/playlist?list=PLCCIAmutGEbd5V3deH6wE1C_yMePvqsGy

Original proposal: https://hackmd.io/@Mesh-Team/BkUohT6gxe

### 4. Details of the Proposal
Mesh offers funding for Gimbalabs contributors to:
- Organise, announce and host biweekly live coding sessions with Mesh
- Record and upload sessions on Gimbalabs/Mesh Youtube
- Coordinate with Mesh to define schedule and agenda of live coding sessions
- Coordinate with Mesh to ensure experienced Mesh devs are present at sessions for live-coding
