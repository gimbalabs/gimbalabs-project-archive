# PPBL 2024 Live Coding: July 29 - August 16

## Gimbalabs Primary Driver
To provide open, replicable safe spaces to learn, to explore, and to empower individuals and organizations anywhere so that we can solve meaningful problems

## Proposal Driver
Code examples and written content for Modules 203, 204, and 205 will be complete by July 31. Creating videos is time consuming, and it is hard to predict what questions people will have about a lesson. We are ready to start live coding again - and it's possible to combine live coding with more efficient + responsive video creation.

## Proposal

### 1. Title or Brief Description
Deliver videos for Modules 203-205 during live coding sessions.

### 2. Background or Rationale
- Videos need to be created for course
- We are delivering live coding sessions for F11 proposal
- Let's build systems that combine the best of live coding and PPBL lessons
- And set a pathway toward continuing improvement of the course

### 3. Intended Outcome
- Videos created and posted to all lessons in Modules 203, 204, 205
- Live sessions

### 4. Details of the Proposal
Run live coding sessions as follows:
- Tuesday 2024-07-30: Module 203
- Wednesday 2024-07-31: Module 204
- Tuesday 2024-08-06: Module 205
- Wednesday 2024-08-07: Invitation to Contribution (301) + Plans for Course Maintenance (governance)

Then, initialize feedback and governance methods for improving the course:
- Andamio content request features
- Gimbalabs course maintenance patterns
