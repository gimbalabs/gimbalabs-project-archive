## Expected Tasks (based on proposal)

The testing phase will span 8 Gimbalabs Open Governance Spaces:
- Newman and Harsha will run 2 hand-holding sessions each (4 sessions total)
- Last 4 sessions will be completely voluntary and open for the community
- Each session host receives 25 ADA compensation
- Total budget: 200 ADA

## Evaluation Criteria Tasks
- [ ] Each week have one or more community members coming forward to host the 15 min governance slot
- [ ] Track votes on dRep community platform
