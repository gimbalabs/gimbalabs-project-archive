# Contributing to Gimbalabs Governance

Welcome! This guide explains how to participate in Gimbalabs governance.

## Ways to Participate

### 1. Join the Conversation

**Discord**
- Join the `#governance` channel
- React to proposals and discussions
- Share your perspective on governance actions

**Open Spaces**
- Thursdays, weekly
- 10-15 minute governance segment
- Live discussion and consent rounds

### 2. Participate in DRep Decisions

When Cardano governance actions are posted:
1. Read the summary in Discord
2. Ask clarifying questions
3. Share your opinion (for/against/concerns)
4. Attend Open Spaces for consent round
5. Raise objections if you have genuine concerns

You don't need to be an expert - your perspective matters.

### 3. Submit a Proposal

Have an idea for Gimbalabs? Submit a proposal:

1. Copy `proposals/TEMPLATE.md`
2. Complete all 11 sections
3. Save to `proposals/draft/your-proposal-name.md`
4. Open a PR or request a collaborator to add it
5. When ready, move to `proposals/open/` to begin consent

**Tips for good proposals:**
- Be specific about what you want to do
- Explain why it matters to Gimbalabs
- Be realistic about resources and timeline
- Consider potential objections proactively

### 4. Review Proposals

When proposals are Open:
- Read the full proposal
- Consider alignment with Gimbalabs values
- Think about impacts (positive and negative)
- Prepare questions or reactions for discussion
- Decide if you have valid objections

### 5. Help with Governance Operations

**Governance role activities:**
- Monitor gov.tools for new actions
- Write summaries of governance proposals
- Facilitate Discord discussions
- Document vote decisions
- Maintain governance repository

Interested? Express interest in Discord or Open Spaces.

## Decision Rights

**Everyone can:**
- Propose ideas
- Participate in discussions
- Raise objections
- Vote in straw polls

**Governance role holder can:**
- Move proposals between states
- Facilitate consent rounds
- Document decisions
- Represent Gimbalabs at governance events

**DRep key holder can:**
- Execute on-chain votes (must match documented decisions)

## Getting Started

1. **Join Discord** - [discord.gg/gimbalabs](https://discord.gg/gimbalabs)
2. **Read recent proposals** - Browse `proposals/` to see examples
3. **Attend Open Spaces** - Get a feel for how discussions work
4. **Ask questions** - No question is too basic

## Principles

When participating in governance, we value:

- **Transparency** - Decisions are documented and public
- **Inclusion** - All community members can participate
- **Experimentation** - "Good enough for now, safe enough to try"
- **Responsibility** - We own our decisions and learn from them
- **Respect** - Disagreement is healthy; personal attacks are not

## Questions?

- Ask in Discord `#governance`
- Raise in Open Spaces
- Open an issue in this repository
